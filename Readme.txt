1. Для использования id в пути файла нужно его указать в соответствующей строке ввода
и нажать кнопку "Сохранить путь".

2. Для  работы с google таблицами в папке storage должен быть файл credentials.json с настройками 
по текстовому файлу credentials.txt. Содержимое файла я не передал, так как git его не пропускает.

3. Для запуска задания каждую минуту нужно на сервере linux в crone задать команду:

* * * * * cd /path-to-your-project && php artisan schedule:run >> /dev/null 2>&1

Для windows настроить планировщик задач
